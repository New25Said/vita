# Vita 26 Hub Web

## Descripción
Vita 26 es un hub web diseñado para estudiantes del instituto, donde se centralizan juegos y aplicaciones. El objetivo es facilitar el acceso a estas herramientas desde un solo lugar, permitiendo tanto jugar en el navegador como instalarlas como apps web.

## Características
- Interfaz clara y minimalista basada en un fondo azul oscuro degradado.
- Secciones diferenciadas para Aplicaciones, Juegos y Utilidades.
- Cada juego o aplicación tiene un cajón con:
  - Icono representativo
  - Banner
  - Breve descripción
  - Botones de acción (Abrir/Jugar e Instalar)
- Logo principal "Vita 26" con diseño colorido y moderno.
- Navegación sencilla y visualmente coherente.

## Estructura del Proyecto
- `index.html` – Página principal con la estructura del hub.
- `styles.css` – Estilos para los cuadros, secciones y botones.
- `assets/` – Carpeta para imágenes, iconos y banners de juegos/aplicaciones.

## Uso
1. Abrir `index.html` en cualquier navegador moderno.
2. Hacer clic en "Abrir" o "Jugar" para ejecutar la app en línea.
3. Hacer clic en "Instalar" para agregar la app al dispositivo como aplicación web.

## Contribuciones
Actualmente es un proyecto personal de prototipo para el instituto. Las sugerencias de diseño o funcionalidad son bienvenidas, pero cualquier cambio debe seguir la estética y la filosofía de simplicidad del hub.
